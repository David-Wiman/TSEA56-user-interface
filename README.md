# User interface

## Prerequisite
To run the project, atleast Python 3.8 is needed.


## Installation
Install [PySide6](https://wiki.qt.io/Qt_for_Python):

```
$ pip install PySide6
```

Clone project:
```
$ git clone git@gitlab.liu.se:da-proj/TSEA56/2022/G08/user-interface.git
```

## Run
To run the project:

```
$ python main.py
```
